# KPC

同时支持 [Intact][1] 和 [Vue][2] 的高质量前端组件库

[文档](https://ksc-fe.github.io/kpc/)

## 安装

```shell
npm install kpc --save
```

## 使用

```js
import {Button} from 'kpc/components/button';

<Button>Hello</Button>
```
