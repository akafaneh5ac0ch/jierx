# 示例

```jsx
import Checkbox from 'kpc/components/checkbox';

<Checkbox>选项</Checkbox>
```

# 属性

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| disabled | 是否禁用 | `Boolean` | `false` |
| value | 复选框组的取值 | `*` | `false` |
| trueValue | 当前复选框选中的值 | `*` | `true` |
| falseValue | 当前复选框取消选中的值 | `*` | `false` |
