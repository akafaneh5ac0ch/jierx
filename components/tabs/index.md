# 标签属性 Tabs
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |

| value | 被选中的标签,使用`v-model`进行双向绑定 | `String`| `undefined`|
| vertical | 垂直排列标签 | - | `undefined` |
