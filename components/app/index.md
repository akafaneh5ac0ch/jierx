# 属性

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| view | 当前要展示的Intact组件实例 | `Intact` | `undefined` |
| container | 当前引用的挂载点 | `HtmlElement` | `undefined` |
| loading | 展示或隐藏loading动画 | `Boolean` | `false` |
| ssr | 当前应用是否进行服务器端渲染 | `Boolean` | `false` |
